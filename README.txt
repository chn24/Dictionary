1. Giới thiệu
	- Dictionary-Master là một từ điển Anh-Việt đơn giản, cũng là bản Java đầu tiên của nhóm chúng mình
	(Đặng Thái Huy và Chu Huy Nghĩa) ở Trường Đại học Công nghệ, Dictionary-Master dựa trên Java, JavaFX và một chút CSS.
	- Hãy nhập những từ, câu, thậm chí là đoạn văn, chúng mình sẽ dịch ra Tiếng Anh hoặc Tiếng Việt ngay 
	lập tức cho bạn
2. Các tính năng
	- Thêm từ mới, chỉnh sửa hoặc xóa bất kỳ từ nào bạn muốn
	- Xuất từ ​​điển
	- Có voice để phát âm mẫu các từ bạn tra cứu
3. Các tính năng mới
	- Làm mới giao diện và trải nghiệm người dùng
	- Lịch sử tra cứu khi bạn thực hiện tra cứu bằng API Google, xóa lịch sử tìm kiếm

4. Cách cài đặt
	-Dictionary-Master là một chương trình mã nguồn mở : https://github.com/chn24/Dictionary
	-Dictionary-Master yêu cầu Java (JDK Amazon Corretto v11.0.12) để chạy
